Embudo de conversión

Usa este recorrido como estructura operativa para no publicar por publicar. Cada pieza debe empujar al siguiente paso.

1. Atracción

Reels, historias, carruseles, hooks, prueba social y promesa clara.

2. Filtrado

Llamada a la acción: palabra clave, formulario o WhatsApp.

3. Educación

NotebookLM resuelve dudas y calienta la intención.

4. Cierre

WhatsApp, audio, link de compra o registro con seguimiento.

Prompts listos para copiar

Aquí tienes comandos operativos para generar contenido, responder prospectos y acelerar cierres.

Prompt de contenido diario

Actúa como estratega y creador de contenido para redes sociales. Mi objetivo hoy es: [VISIBILIDAD / ENGAGEMENT / CAPTACIÓN]. Mi nicho es: [NICHO]. Mi oferta principal es: [OFERTA]. Genera: 1. Un hook fuerte 2. Un guion corto para reel 3. Un copy para post 4. Un CTA que lleve a WhatsApp 5. Una idea de historia para reforzar el mensaje Hazlo directo, persuasivo y orientado a conversión.

Prompt de respuesta a objeciones

Actúa como cerrador profesional. Responde la siguiente objeción de forma segura, simple y persuasiva: [PEGA AQUÍ LA OBJECIÓN] La respuesta debe: 1. Validar la duda 2. Quitar fricción 3. Reforzar el valor 4. Llevar a una acción concreta Evita sonar necesitado. Suena firme y elegante.

Prompt de cierre por WhatsApp

Actúa como experto en ventas por WhatsApp. Crea 3 respuestas de cierre para un prospecto que ya mostró interés en [PRODUCTO / NEGOCIO]. Las respuestas deben tener estos tonos: 1. Cercano y humano 2. Seguro y directo 3. Premium y profesional Cada una debe terminar con una invitación clara a comprar, registrarse o agendar.

Prompt de seguimiento

Actúa como experto en seguimiento comercial. Escribe 5 mensajes de seguimiento para una persona que pidió información pero no respondió. Condiciones: - No sonar intenso - Generar curiosidad - Recordar valor - Invitar a retomar la conversación Dame versiones para día 1, día 2, día 4, día 7 y día 10.
